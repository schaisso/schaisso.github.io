Arrays are Objects that are ordered lists of elements separated by commas. An array creation looks like:

var myArray = [element0, element1, ..., elementN];

or like:

var myArray = new Array();

The first array would look like:

var myArray = [ 'element0' 'element1' ... 'elementN' ]

Arrays are ordered, and stay in the order they are created in. The array [1, 2, 3] is different from the array [2, 1, 3]. Elements can be added to an array directly at specific indices, but it's best not to, although accessing them that way is okay. Adding elements directly inserts undefined into indices not already used (ex: 1, 2, undefined, 4). Array indices are strings that can be accessed by array-style notation using brackets, and any numbers used for an array index will be coerced into strings. JS won't complain if you try to index an array with a variable whose contents don't represent a number, and rather than adding an index to the array, you'd add a property to that array, which is just a named piece of information that we can only get with reference to the Object. Ex: Array.length. In JS, we can assign properties to arrays. Ex: array.myProp = "I'm a property"

Some common array manipulations are:

Mutators(change the underlying array):
push: adds an element to the end of the array
pop: removes an element from the end of the array
unshift: adds an element to the beginning of the array
shift: removes an element from the beginning of the array

Non-destructive and tools(preserve the underlying array):
spread operator: three dots in a row, spreads out the arrays contents, copies the contents that are in the underlying array, can be used to make a new array to manipulate.
slice: takes a slice from the array. The first argument specifies start point, second argument (if any) specifies end point. Passing a negative (-n) argument to slice results in the last n elements in an array. Slice can be strategically used to avoid mutating the array in ways that push/pop/unshift/shift would do.
splice: useful in removing elements from the middle of an array. Splice takes an index in the array as the first argument, the number of elements to remove as the second, and any number of elements to add after the second. With no arguments, splice returns an empty array and doesn't affect the target array. Examples
